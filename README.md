# Flow — Project Management
A stunning, high-performance, purely client-side SaaS project management application with zero external dependencies.

## Features
- 📊 **Overview Dashboard**: Live stats, burn-down reports, and radial progress charts via native CSS.
- ☷ **Kanban Board**: Drag-and-drop enabled status workflows.
- ⊞ **Main Table**: High-density task lists with immediate inline editing capabilities.
- ⟶ **Gantt Timeline**: A scalable sprint calendar visualized through DOM.
- 🌳 **Hierarchy Tree**: Organization visualizer.

## Tech Stack
- Frontend: Pure HTML, Vanilla JS, Vanilla CSS.
